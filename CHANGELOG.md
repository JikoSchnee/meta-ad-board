# 改动日志

用于记录每次功能更新的标题和内容，方便整理 commit message。

## 2026-06-02

### 初始化 Meta Ads 数据分析看板

内容：
- 新增 Streamlit 应用入口 `app.py`，支持上传 Meta Ads Manager 导出的 CSV、Excel 文件。
- 支持中文字段和部分英文别名识别，包括日期、花费、展示、点击、加购、结账、购买和 ROAS。
- 实现数据清洗、数值字段标准化、日期解析、每日聚合和核心指标计算。
- 新增 KPI 总览，展示总消耗、展示/点击、整体 CTR、购物 ROAS 和 CPA。
- 新增趋势图、效率图、转化漏斗、广告层级排行、明细表和每日聚合 CSV 下载。
- 新增自动化诊断建议，用 CTR、加购、结账、购买、ROAS、CPC 等指标给出投放优化方向。
- 新增 `requirements.txt` 和 `README.md`，说明依赖安装、启动方式和建议输入字段。

### 支持局域网访问

内容：
- 通过 Streamlit 的 `--server.address 0.0.0.0` 启动方式，让同一局域网设备可以访问应用。
- 确认当前局域网访问地址为 `http://172.18.169.25:8502`。
- 保留本机访问方式，方便同时用 `localhost` 调试。

### 添加广告指标缩写悬停解释

内容：
- 新增统一缩写词表，维护 CTR、CPC、CPM、CPA、CVR、ROAS、KPI 的中英文全称。
- 在 KPI 卡片和图表标题中将缩写渲染为可悬停提示文本。
- 新增 tooltip 样式，鼠标悬停时展示如“点击率 / Click-Through Rate”。
- 将部分 `st.subheader` 替换为自定义标题渲染，支持标题内嵌缩写解释。

### 增强广告转化漏斗与 CPA 指标

内容：
- 将漏斗起点调整为链接点击量，更聚焦广告点击后的电商转化链路。
- 新增加购次数/加购率、发起结账次数/发起结账率、购物次数/购物完成率。
- 新增单次成效费用 CPA 指标，按总花费除以购物次数计算。
- 在漏斗图标签中展示各阶段转化率，便于定位点击、加购、结账和购物之间的流失。

### 调整漏斗主链路与转化率展示

内容：
- 将漏斗图节点改为纯数量链路：链接点击量、加购次数、发起结账次数、购物次数。
- 将加购率、发起结账率和购物完成率移到旁边指标卡展示，避免与主漏斗节点混写。
- 将漏斗图自身的百分比文字移除，主图仅展示各阶段数量。
- 保留 CPA 单次成效费用卡片，继续用于评估购物成效成本。

### Apple 风格重设计广告转化漏斗

内容：
- 使用 awesome-design-md 的 Apple 设计参考，重塑漏斗区为白底、浅灰画布、18px 圆角、细 hairline 和单一蓝色强调。
- 将广告转化漏斗拆成两张图：含加购路径和无加购路径。
- 含加购路径展示链接点击量、加购次数、发起结账次数、购物次数。
- 无加购路径展示链接点击量、发起结账次数、购物次数，便于查看直接结账链路。
- 将加购率、发起结账率、购物完成率和 CPA 改为独立指标卡，并使用轻量进度条表达比例。

### 改为切换显示加购环节

内容：
- 将两张并排漏斗图改为单张漏斗图，通过“含加购环节 / 无加购环节”切换显示。
- 切换为含加购环节时，漏斗显示链接点击量、加购次数、发起结账次数、购物次数。
- 切换为无加购环节时，漏斗显示链接点击量、发起结账次数、购物次数。
- 右侧转化率随切换同步变化：含加购时发起结账率按结账次数 / 加购次数计算，无加购时按结账次数 / 链接点击量计算。

### 增加 CPA 变化表与自定义漏斗环节

内容：
- 在趋势效率页新增单次成效费用 CPA 的每日变化折线图和明细表。
- 新增“点击量”字段识别，和“链接点击量”分开用于漏斗分析。
- 将转化漏斗改为多选环节，支持从点击量、链接点击量、加购次数、结账次数、购买次数中选择多个内容。
- 根据用户选择的环节按固定业务顺序生成漏斗，并自动计算每一步到下一步的转化率。
- 使用自定义 Apple 风格漏斗组件，把相邻环节连接起来，并在连接线中间展示转化率百分比。

### 改为 Plotly 漏斗图并新增落地页浏览量

内容：
- 新增“落地页浏览量”字段识别，支持 Landing page views 等英文别名。
- 将转化漏斗选项调整为点击量、落地页浏览量、加购次数、发起结账次数、购买次数。
- 默认漏斗路径改为点击量、落地页浏览量、发起结账次数、购买次数，不默认包含加购次数。
- 使用 Plotly 漏斗图渲染用户选中的环节，避免 HTML 原文显示问题。
- 漏斗图每个环节显示具体数字和相邻环节转化百分比。

### 修复漏斗转化率与点击量回退

内容：
- 不再使用 Plotly 内置 `percent previous`，改为手动计算相邻环节转化率，避免所有环节都显示 100%。
- 第一环显示为“入口”，后续环节显示当前环节数量除以上一环节数量的百分比。
- 当数据没有“点击量”但有“链接点击量”时，自动用链接点击量回填点击量，避免漏斗入口为 0。

### 新增自定义数据对比页

内容：
- 新增“自定义对比”页签，支持选择多个每日指标在同一张折线图中对比。
- 支持花费、展示、覆盖、点击量、链接点击量、落地页浏览量、加购、结账、购买、CTR、CPC、CPM、CVR、CPA、ROAS 等指标。
- 提供标准化对比和原始数值对比两种模式，便于不同量级指标放在同一张图中观察趋势。
- 在图表下方展示所选指标的每日明细表。

### 适配视频播放报表与广告名称对比

内容：
- 新增视频播放量、单次展示的播放视频达3秒率、视频平均播放时长三个字段识别。
- 当源表同时包含上述三个视频字段时，在自定义数据对比中自动增加对应指标。
- 视频播放量按总量汇总，3秒播放率按展示次数加权，平均播放时长按视频播放量加权。
- 自定义数据对比新增“总量对比 / 按广告名称对比”开关。
- 按广告名称对比时，可选择多个广告名称，并按日期展示不同广告名称的指标趋势。

### 覆盖上传表全部数字指标
